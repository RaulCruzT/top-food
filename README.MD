## Development

Top Food was developed using:

* Vite
* React
* TypeScript

## Installation guide

1. Download the project
2. Inside top-food folder open a terminal and type:
    ```
    npm install
    ```
3. Press enter

## Execution guide

1. Inside top-food folder open a terminal and type:
    ```
    npm run dev
    ```
